# ESP32-iot
### 前言：
    
    我们经常有这种烦恼，想要养宠物但是奈何工作繁忙没时间给宠物喂食、没时间照顾它们，每天只有早上和晚上能见到小可爱们。所以我有了可以随时看到宠物、可以远程喂食、照顾宠物的想法。
    
### 前置：

准备以下材料：

1. ESP32 主控板
2. SG90 舵机
3. 5V 继电器模块
4. DHT11 温湿度传感器模块
5. 面包板
6. 杜邦线
7. 杜邦线若干
8. 加湿器
